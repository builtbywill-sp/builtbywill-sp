## â±ï¸ React Hook: useEffect

### Mount only
```js
useEffect(() => {
  console.log("ONLY on mount");
}, []);
```

### Mount + `showModal` changes
```js
useEffect(() => {
  console.log(`on mount AND on showModal change`);
}, [showModal]);
```

### Better version (Visual feedback):
```js
const [log, setLog] = useState("");

useEffect(() => {
  setLog("Modal was shown!");
}, [showModal]);

return <p>{log}</p>;
```
